# 简介

这是一个QT C++程序，它能够帮助你自动切换音频设备

开发背景：我的电脑连接了许多的音频设备，在我打游戏的时候，我希望它可以自动切换到耳机，在我看视频听歌的时候，我希望它切换到音响。

# 速食手册

2. 其次选择对应的音频设备
3. 点击“添加绑定”即可
